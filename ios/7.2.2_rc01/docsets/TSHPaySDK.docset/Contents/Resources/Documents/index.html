Welcome to Thales Gemalto TSH Pay Documentation

TSH Pay SDK cloud platform

The Trusted Services Hub (TSH Pay) cloud platform is used to digitise any payment card of any brands into your banking application or payment wallet. These digitised cards can be used seamlessly for contactless and QR Code based transactions.

It comes with a cloud offer and a payment SDK or a white label wallet.

The Thales Gemalto TSH Pay TSH Pay cloud service is connected with the major international and domestic payment networks. It currently supports both Mastercard and Visa, along with the white label payment scheme framework called PURE.

TSH Pay SDK

Using the TSH Pay solution, there are two ways in which the Wallet providers are able to implement a token requestor—via the SDK, or the white label wallet.

TSH Pay SDK can either be integrated into your mobile application, or it can be used in the off-the-shelf wallet for:

  • Card enrolment
  • Identity verification
  • Payment keys replenishment
  • Token lifecycle management
  • Plus proximity and in-app payment

As international schemes require certifications, TSH Pay SDK includes all the required security to ensure that you are in compliance with the payment scheme directives.
