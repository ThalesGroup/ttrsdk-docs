  • KnownCpsErrorCodes

    The list of known status reasons from server response.
    The list does NOT include the “System Error” from server, which includes 670 to 679.

    Warning

    The list is for reference purpose. The server may send a status reason that is not priorly known. In typical use case, the MPA shall only need to propagate these errors to the user (with the advice to contact customer support and providing the error code). The SDK may decide from time to time to manage some of these errors internally and whether or not propagate it to the MPA.
